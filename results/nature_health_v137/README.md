# Nature Health v137 derived outputs

This directory contains releasable derived outputs for the Nature Health v137 submission:

**Coverage without convergence: Geographic inequality in proximity to AI-reporting hospitals in the United States, 2022-2024**

These files are derived from licensed AHA hospital data linked to public Census and County Health Rankings data. Raw AHA source records are not redistributed.

## Figures

- `figures/2022_vs_2024_diffusion_v137.png`: main-text Figure 1.
- `figures/manuscript_travel_distribution_ai_rucc.png`: main-text Figure 2.
- `figures/2022_2024_delta_map_v137.png`: main-text Figure 3.
- `figures/appendix_ai_flag_mode_sensitivity_2024_v137.png`: supplementary AI-flag sensitivity figure.

## Tables

- `tables/table1_longitudinal_summary_v137.csv`: main-text Table 1.
- `tables/table2_blockgroup_summary_v137.csv`: main-text Table 2.
- `tables/table3_transition_equity_profile_v137.csv`: main-text Table 3.

All table values are aggregated manuscript outputs and do not disclose restricted hospital-level AHA records.
