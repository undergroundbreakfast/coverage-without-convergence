# Code

`geospatial_access_workflow_v137.py` documents reusable public calculations and validates the releasable v137 outputs for the Nature Health access manuscript.

The manuscript's core analyses use proximity, population coverage, Lorenz/Gini, map, and threshold-analysis logic. The full end-to-end workflow requires licensed AHA source records and local geocoded hospital tables that cannot be redistributed here, so this public script focuses on reusable methods and output validation rather than raw-data extraction.

Raw AHA files, local database dumps, and companion-project mortality/TMLE/propensity scripts are intentionally excluded.
